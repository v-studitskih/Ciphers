# Chipers
Проект реализует несколько классических алгоритмов шифрования
Шифр Цезаря - простой подстановочный шифр со сдвигом алфавита

Шифр Виженера - полиалфавитный шифр на основе ключевого слова

RSA - асимметричный криптографический алгоритм

RC5 (только 32-битная версия) - симметричный блочный шифр

Проект разработан в образовательных целях для демонстрации основных принципов криптографии.
